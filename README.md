# Express-todo

## Pre requirements
* [Node.js](https://nodejs.org/en/)
* [NPM](https://www.npmjs.com/get-npm)

## Clone
```
git clone https://github.com/Brenin1991/express-todo.git
```
## Install dependencies
```
npm install
```
